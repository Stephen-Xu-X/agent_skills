## 章节说明

| 章节 | 用途 |
|------|------|
| `📋 今日待办` | 待办清单，已完成用 `[x]`，未完成用 `[ ]`，备注缩进在下方 |
| `💬 碎碎念` | 口语化感想、吐槽、随机想法，每条用 `- ` 开头 |
| `🌙 夜晚复盘` | 晚间回顾 |
| `🫵 心情波动` | 当天的心情/情绪记录 |
| `✅ 今日收获` | 学到的、做成的、有成就感的事 |
| `⏩ 明日方向` | 第二天要做的事（会被自动搬运） |

## Todo 格式规范

单行任务：
```
- [ ] 具体任务
```

带备注的任务：
```
- [ ] 具体任务
      备注内容
      另一条备注
```

已完成的任务：
```
- [x] 已经完成的事
```
