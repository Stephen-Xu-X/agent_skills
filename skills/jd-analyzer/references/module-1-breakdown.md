# Module 1: 岗位结构化拆解

**开场概括**（1段）：
> "这是一家[公司类型]的[岗位名称]职位，整体是个[层级]岗，核心在于[1-2句]。JD写得[详细/模糊/有亮点]，帮你拆开来看……"

**1. 硬性门槛（Must Have）**
学历、年限、必备技能/工具、行业经验要求

**2. 加分项（Nice to Have）**
"优先考虑"类条件、额外技能、特定背景偏好

**3. 核心职责分解**（表格，按重要性排序）

| 优先级 | 职责描述 | 做什么 | 交付什么 |
|--------|----------|--------|----------|
| ★★★   | …        | …      | …        |
| ★★☆   | …        | …      | …        |

**4. 🚦 岗位红绿灯信号**

从JD措辞中挖掘隐性信号：
- 🟢 **正向**：成长空间、资源支持、团队重视程度
- 🟡 **需关注**：模糊描述、潜在挑战、不确定因素
- 🔴 **风险**：高压文化、执行属性过强、定位不清
